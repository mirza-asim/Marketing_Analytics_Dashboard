# Marketing_Analytics_Dashboard
Business Data Analysis of Marketing Data

Just completed an end-to-end Marketing Analytics Dashboard project where I focused on turning raw data into actionable business insights 📊

This project was built to simulate real-world decision-making and demonstrate how data can directly impact profitability and strategy.

🔍 What I worked on:

Cleaned and transformed raw datasets using Power BI (Power Query)

Wrote SQL queries to extract and structure relevant business data

Built interactive dashboards in Power BI

Used DAX to create KPIs such as Conversion Rate, Customer Sentiment Score, and channel performance metrics

Performed sentiment analysis using Python to understand customer feedback trends

📈 Key highlights:

Compared performance across marketing channels (Email, Social, Ads)

Built a funnel visualization to identify drop-off points

Identified areas where cost optimization and conversion improvements could increase revenue

Delivered a drill-down analysis with clear, data-driven recommendations

💡 Outcome:

The project doesn’t just visualize data, it answers “what should the business do next?” by highlighting optimization opportunities across campaigns and customer journeys.

This experience helped me strengthen:

Data analysis & storytelling

Business problem solving

Dashboard design & KPI tracking
